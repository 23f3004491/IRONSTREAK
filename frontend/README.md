# IronStreak Frontend

Minimal Next.js (App Router) + Tailwind frontend.

## Setup

```bash
cd frontend
npm install
npm run dev
```

The dashboard includes login, creating targets, and checking in daily.

For deployment, set `NEXT_PUBLIC_BACKEND_URL` to your deployed FastAPI backend.
